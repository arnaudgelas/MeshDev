MeshDev Readme
Mesh Comparison Software
Copyright (C) 2002-2003 Michaël Roy


MeshDev is a mesh comparison software. It accepts two meshes as input.
MeshDev return numerical values and optionaly visual results (with a
pseudo-colored mesh corresponding to the measured deviation)

For more information about the technics used in this software, read papers
available on the MeshDev web site: 
http://meshdev.sourceforge.net
